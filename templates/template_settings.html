Template Guide

Templates are Word .dotx files that serve as the master layout for generated documents. When you download a test record or SOP, the system clones the template and fills in the data from the database. The original template is never modified.

Expected templates:

  • PK Specification Test Record Template.dotx — For packaging spec test records
  • RM Specification Test Record Template.dotx — For raw material spec test records
  • SOP Temp.dotx — For Standard Operating Procedures
  • CLB003 Component Receiving Record.docx — For component receiving records (pre-filled per spec)
  • CLB008 QC Release Sticker.docx — QC Lab Release stickers (30 per page)
  • CLB009 QC Sampled.docx — QC Lab Sampled stickers (30 per page)
  • CLB010 Component ID Tag.docx — For component ID tags (label sheet, 6 per page)
